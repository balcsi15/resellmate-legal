# ResellMate Privacy Policy

Effective date: June 1, 2026

Developer/contact: Alex Balazs — balazs.alex90@gmail.com

Published privacy policy URL: [https://balcsi15.github.io/resellmate-legal/](https://github.com/balcsi15/resellmate-legal)
## Overview

ResellMate helps second-hand sellers create honest listing text from real item photos and seller-provided item details. The app is designed for Vinted-style listings for clothing, shoes, bags, accessories, and similar items.

ResellMate is not affiliated with Vinted, OLX, eBay, Depop, Facebook Marketplace, or any other marketplace.

## Data ResellMate Processes

When you create listing text, ResellMate may process:

- item photos you choose to upload for that listing-analysis request;
- seller-entered item details such as category, brand, size or model, condition, color, flaws, material, measurements, notes, and target price;
- generated listing text, title options, checklists, price guidance, warnings, and listing score;
- an anonymous `clientInstallationId` generated by the app to show saved history on the same app installation.

The `clientInstallationId` is not an account, login, authentication method, or security boundary.

ResellMate does not currently process account credentials, payment details, advertising identifiers, contacts, precise location, or marketplace account credentials.

## How Data Is Used

ResellMate uses uploaded item photos and seller-entered details to generate listing guidance, including titles, descriptions, missing-photo checklists, measurement prompts, flaw-disclosure prompts, verification notes, and approximate price guidance.

Generated output is a starting point only. Sellers must verify all generated details before posting a listing. ResellMate must not be used to hide flaws, invent brand, material, condition, authenticity, provenance, included accessories, or marketplace prices.

## Backend and AI Processing

Uploaded photos and seller-entered details are sent to the hosted ResellMate backend during the listing-analysis request.

When AI generation is enabled, ResellMate may send seller-provided details and generated listing context to OpenAI for AI-backed listing generation. If AI is unavailable, ResellMate falls back to deterministic non-AI output.

Do not upload private documents, IDs, payment cards, or photos that are not needed to describe the item being sold.

## Storage and Retention

The current MVP stores generated listing history metadata and generated listing text in PostgreSQL/Neon so the app can show saved history for the same anonymous installation.

Original uploaded item photos are not saved in history in the current MVP. They are processed for the request and are not available on saved detail screens.

There are no user accounts yet, so ResellMate does not currently provide account-based history ownership, account deletion, or cross-device history.

Generated history deletion is not implemented yet. This policy should be updated if account deletion, manual history deletion, or persistent original image storage is added.

## Data Sharing and Third-Party Processing

ResellMate does not sell user data.

Data may be processed by:

- the hosted ResellMate backend;
- the managed PostgreSQL database provider used for generated listing history;
- OpenAI when AI-backed listing generation is enabled.

ResellMate does not directly integrate with, scrape, or auto-post to Vinted, OLX, eBay, Depop, Facebook Marketplace, or other marketplaces.

ResellMate currently does not include ads, analytics SDKs, payment processing, subscriptions, or marketplace credential handling.

## Current Non-Features

The current MVP does not include:

- user accounts or authentication;
- payments or subscriptions;
- ads or analytics;
- persistent original image storage;
- marketplace posting or marketplace automation;
- direct marketplace integrations;
- automatic listing publication.

## Security Notes

Data is transmitted to the hosted backend over HTTPS when the hosted API is used. The app should not include API keys, database connection strings, signing passwords, or other secrets.

## Changes

This policy should be updated when ResellMate changes its data practices, including adding accounts, payments, persistent image storage, analytics, marketplace integrations, or public Play Store release support.
